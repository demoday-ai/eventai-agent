# EventAI - AI-агент-куратор Demo Day

## Контекст

MVP системы EventAI разработан на [AI Talent Camp 2026](https://github.com/demoday-ai/demoday-core) (недельный интенсив в рамках магистратуры AI Talent Hub, ИТМО). Система состоит из Telegram-бота с AI-агентом, веб-админки для организаторов и общего бэкенда (FastAPI + PostgreSQL + Qdrant).

**В рамках бонус-трека LLM & Agentic AI реализуется агентская часть системы** - Telegram-бот с диалоговым профилированием, рекомендательным пайплайном и инструментами агента. Веб-админка и инфраструктура оркестрации описаны как окружение агента, но не являются предметом данного проекта.

- Репозиторий бонус-трека: [dpGorbunov/bonus-track-llm](https://github.com/dpGorbunov/bonus-track-llm)
- Основной репозиторий: [demoday-ai/demoday-core](https://github.com/demoday-ai/demoday-core) (приватный, доступ по запросу)

## Задача

Demo Day AI Talent Hub (ИТМО) - 330 проектов, 10 залов, 2 дня. Проблемы:

- Гость видит ~16% программы и выбирает вслепую
- Эксперты приходят без информации, не могут выбрать слоты, оценки собираются в таблицах
- Вопросы участников летят в личку организатору - нет единого канала

## Целевая аудитория агента

- **Внешние гости** (партнеры, инвесторы, HR, менторы) - не могут найти релевантные проекты среди сотен
- **Эксперты** (~50 чел.) - нет инструмента для выбора слотов и выставления оценок
- **Бизнес-партнеры** (~15-20 чел.) - нет поиска проектов под задачу, постсобытийное сопровождение хаотичное

## Что делает агент

- **Профилирование:** LLM уточняет роль/интересы/цели за 2-3 реплики - структурированный профиль
- **Рекомендации:** эмбеддинг (Gemini) - Qdrant top-30 - переранжирование с учетом расписания - LLM-резюме - top-15
- **Анализ артефактов:** парсинг презентаций (PPTX/PDF) и GitHub-репозиториев; агент отвечает по содержимому проекта
- **Вызов инструментов:** карточка проекта, сравнение 2-5 проектов, подсказки для Q&A, пакет контактов, воронка для бизнеса, вопрос организатору
- **Эксперты в боте:** выбор слотов доступности, оценки проектов после защиты
- **Канал бот-админка:** вопросы участников - админка - ответ - бот
- **Плавная деградация:** LLM недоступна - ответы без LLM, Qdrant недоступен - пересечение тегов, ключ заблокирован - ротация + запасная модель

## Окружение агента (не в рамках бонус-трека)

Агент работает внутри системы EventAI, которая включает:

- Веб-админку организатора (импорт данных, кластеризация, расписание, рассылки)
- Общий бэкенд (FastAPI, 87 эндпоинтов, PostgreSQL, Qdrant, Celery)
- Инфраструктуру (Yandex Cloud, Traefik, RabbitMQ)

Эти компоненты разработаны на AI Talent Camp и используются агентом через API.

## За рамками PoC

- Организация встреч 1:1 (только запрос контакта автора)
- Самостоятельная загрузка артефактов студентами
- Автоматическая фильтрация персональных данных
- Ограничение частоты запросов к боту
- Локальная LLM
